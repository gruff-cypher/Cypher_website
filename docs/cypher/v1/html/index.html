弊社だからこそ提供できる価値

運営会社、株式会社グラフとは

グラフは「データの利用価値を最大限に拡張する」ことをミッションとしたデータサイエンスAI(人工知能)企業です。

各産業界トップ企業での実績を持ち、データ利活用において実際的なノウハウを蓄積してきました。

AI(人工知能)・機械学習の活用やビッグデータ分析を通じて、クライアントにとって本当に必要な支援、ビジネス戦略コンサルティングを提供しています。

代表紹介

代表紹介: 原田博植

日経データサイエンティストオブザイヤー受賞
代表の原田は、毎年1人だけ選出される日本経済新聞社主催「データサイエンティスト・オブ・ザ・イヤー」を2015年に受賞した、日本トップクラスのデータサイエンティストです。
シンクタンク、外資ITベンチャー、リクルートにてAI・機械学習の事業装着と組織設計をゼロから起案・具現化し、リクルート初のチーフデータサイエンティストとなりました。

多彩なイベント

CYPHERでは、現役データサイエンティストとのキャリア相談会や交流イベントなどを定期的に開催します 。

よくある質問 (FAQ)

CYPHERへの応募資格は何ですか？

詳細な応募資格は募集ページより募集要項をご確認ください。主にデータサイエンスに興味のある学生、またはデータサイエンス関連のスキルを向上させたい方が対象です。

プログラムの期間はどれくらいですか？

プログラムの期間は参加する案件やコンペによって異なりますが、柔軟に対応できるよう調整可能です。

未経験でも参加できますか？

LADプログラムを通じて基礎から学べるため、意欲があれば未経験の方でもご応募いただけます。サポート体制も充実しています。

選考プロセスについて教えてください。

通常、書類選考と面接を行います。詳細については、応募後にご案内いたします。

あなたの未来を、CYPHERで築きませんか？

一期生として、ここでしか得られない実践的な経験と仲間との出会いを掴みましょう。
